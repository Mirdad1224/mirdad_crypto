
# Cryptocurrency Project

This project uses API to display different cryptocurrency coins and their charts.


## Tech Stack

React, MaterialUI, Chartjs2, Swiper



## Features

- Light/dark mode toggle
- Search
- Differnt charts
- Responsive Design
